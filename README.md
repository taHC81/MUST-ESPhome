# MUST-ESPhome
This project aims to make a wireless monitor of MUST brand inverter, developed and tested with PV1800 model. It uses the ESPhome to intergate inverter into Home assistant.

## HW prototype
- Wemos D1 mini Pro
- RS485-UART adapter
